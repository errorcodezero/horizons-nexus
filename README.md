# Horizons Nexus Project

- Operation Crux: Bomb Sydney, Australia - "Look! There's something on the horizon." (upside-down)
- Operation Arcana: Bomb Singapore, Singapore - "தொடுவானத்தில் ஏதோ ஒன்று தெரிகிறது"
